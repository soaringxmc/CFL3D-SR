CFL3D Tutorial

Course Notes from a tutorial on CFL3D usage are available as a NASA Technical Memorandum (NASA/TM-2006-214301, April 2006), titled "CFL3D Version 6.4 - General Usage and Aeroelastic Analysis." This document summarizes many of the newest features of the code (as of Version 6.4), and also explains in detail how to set up and run it for general cases. Particular emphasis is given to CFL3D's upgraded deforming mesh and aeroelastic analysis capabilities.

The target audience ranges from basic to advanced users. New users should find useful the discussion of general features of the code and the many options that are available, code set-up description, grid-characteristics recommendations, and input requirements. Advanced users should find useful the discussion on new code features, keywords, and discussion on unsteady computations and on parallel (MPI) usage. Although they are generally useful for any CFL3D user, these notes are particularly recommended (if not crucial) for users interested in using CFL3D for deforming mesh and/or aeroelastic analysis.
