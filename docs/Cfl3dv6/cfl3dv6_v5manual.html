Downloadable Files

    There is no manual available for CFL3D Version 6. We recommend that you use the Version 5 manual given here in combination with the information on the New Features page.

  • Entire 354-page manual as a tar file:

    CFL3D Version 5.0 Manual (5.3Mb)

    The file un-tars as a directory called V5Manual, with all 26 individual-chapter pdf files below, plus the title-page file and the corrections file. Last update 8/7/09; the tar file is updated when there is any change to one or more of the individual files).

    Note: virtually all of the information in the Version 5 manual is applicable to Version 6; where differences occur, they are noted in the Summary of Input File Differences section of the New Features page.
